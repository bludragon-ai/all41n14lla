# Origin

The name comes from *My Hero Academia*. One For All is the quirk Deku inherits from All Might — a power that accumulates strength from every previous holder, each one passing forward what they built. All For One is its mirror: the antagonist's ability to take quirks from others. I built a memory system, so the metaphor landed hard. Memory inherited across agents. Each agent passing what it learned to the next. The user as the stable identity that all of those agents serve. If you know the show, you know the resonance. If you don't, the short version is: a power that grows because each holder adds to it.

The stylization — `all41n14lla` — is "all 4 1 n 14 lla", which is "all for one and one for all" with numbers subbed in for letters. `4` for "for", `1` for "one", `14` for "and" (one-four → "and"), then the palindrome flips and closes with `lla` (the reverse of "all"). It is unreadable at first glance. That is part of the joke. Once you see it, you cannot unsee it, and the name becomes a handshake — the people who get it, get it. Branding by in-group recognition rather than by marketing polish.

I considered rebranding to `mindpalace` before the first release. Cleaner. More memorable. Says what it is without needing a decoder ring. But I kept `all41n14lla` for v0.1 because the codename *is* the hook. The project has a personality, not a marketing name. A clean, memorable name is what you pick when you are trying to sound professional; a name that looks like a keyboard smash is what you pick when the work is supposed to do the talking. `mindpalace` is reserved — it becomes the name of the future paid tier, when there is one. For now, this is the memory system with the weird name. That is correct for v0.1. I built this because agent memory is broken, and fixing it was more important than picking a name that would poll well.
